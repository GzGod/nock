# 配置要求
最低要求 64 G RAM 没有到这个要求的可以省省了。。。

## 一键命令 Linux 版
   ```bash
wget -O nock-install.sh https://raw.githubusercontent.com/GzGod/nock/refs/heads/main/nock-install.sh && sed -i 's/\r$//' nock-install.sh && chmod +x nock-install.sh && ./nock-install.sh
   ```
